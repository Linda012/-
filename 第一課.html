N5第一課文法

第一課


A は B です (A是B)

は：助詞，讀音為 wa

です：表肯定

A は B ではありません (A不是B)

ではありません：表否定

ではありません ＝ じゃありません＝ではないです

では＝じゃ

ありません＝ない

A は B ですか？ (A是B嗎?)

か：疑問助詞

  • Yes --> はい、A は B です
  • No --> いいえ、A は B ではありません

    • *這裡的 A,B 皆放入名詞

    敬體與常體之間肯定與否定的差異

    敬體 常體
    肯定 はい、A は B です。 はい、A は B だ。
    否定 いいえ、A は B ではありません。 いいえ、A は B ではない。
    いいえ、A は B ではないです。
    否定(較口語) いいえ、A は B じゃありません。 いいえ、A は B じゃない。

    指示詞

    在此假設自己為說話者，另一個人為聽話者，在有點距離的情況下。


    用法 意思 距離 解釋
    これ　この　ここ 這 近距離 離自己較近的事物或場所
    それ　その　そこ 那 中距離 離對方較近的事物或場所
    あれ　あの　あそこ 那 遠距離 離雙方都很遠的事物或場所
    どれ　どの　どこ 哪一個 距離不確定 指三件以上人事物之其中一個

  • これ／それ／あれ　＋　は　＋　N　＋です
  • これ／それ／あれ　＋　は　＋ S　＋　の　＋　N　＋　です
  • この／その／あの　＋　N１(物）＋　は　＋　 S　＋　の　＋（N1）＋　です
  • N　＋　は　＋　どれ　＋　ですか
  • N　＋　は　＋　どの　＋　N　＋　ですか

    • 例：図書館はどの建物ですか。

    ★

    どれ問　→　これ／それ／あれ答

    どの問　→　この／その／あの答

    依照距離提問語回答

  • 物品物品兩人皆靠近：問：これ／この　→　答：これ／この
  • 物品離A有段距離，離B比較近：
    A問：それ／その　B答：これ／この
  • 物品離兩人都遠：問：あれ／あの　→　答：あれ／あの

    • 依照距離提問語回答

  • 自己&對方所在的場所：問：ここ　→　答：ここ
  • 離A有點距離的場所，離B較近：
    A問：そこ　B答：ここ
  • 離自己&對方都遠的場所：問：あそこ　→　答：あそこ

    • 助詞の

    N1 の N2 (某人的東西)

    N1為人，N2 為事物

    助詞も

    N1 も N2 です(某人也是...)

    表示並例，中文"也"的意思。
